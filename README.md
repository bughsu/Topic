# Qt6 專業多路監控錄影系統 - 智慧移動偵測

專業的多路視訊監控系統，整合 OpenCV 智慧移動偵測和完整的事件日誌功能。

## ⭐ 核心功能

### 智慧移動偵測
- 即時影像分析，使用 OpenCV MOG2 背景減除算法
- 當畫面像素變化超過閾值時自動觸發事件
- 可調整靈敏度（1%-10%）
- 自動儲存移動偵測快照

### 事件日誌系統
- 完整記錄所有系統事件
- 移動偵測、錄影、串流連接等事件追蹤
- JSON 格式持久化儲存
- 事件瀏覽和管理介面

### 多路監控錄影
- 同時支援多個串流來源
- 支援 RTSP、HTTP、MJPEG 等格式
- 偵測到移動時自動錄影（可選）
- 使用 FFmpeg 高效錄影

## 📋 系統需求

- Qt 6.x
- FFmpeg 8.0.1+
- OpenCV 4.x
- C++17 編譯器

## 🚀 快速開始

1. 安裝依賴項（Qt6、FFmpeg、OpenCV）
2. 設定 `Topic.pro` 中的路徑
3. 使用 Qt Creator 建置專案
4. 執行程式並新增串流來源
5. 啟用移動偵測功能

- **監控畫面**：九宮格多路顯示
- **移動偵測控制**：靈敏度調整、自動錄影選項
- **事件日誌**：查看所有記錄的事件
